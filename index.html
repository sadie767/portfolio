Description of Projects

Dream Vacation
In this project we were required to create a site that displayed our dream vacation. Using HTML and CSS the tasks were tailored to Bootstrap and included; creating containers, using a Jumbotron, using blockquotes, and using helper classes.
Justine's Resort
In this project we were required to create a site that advertised a resort. Using HTML and CSS the tasks were to have a main home page that linked to 5 different webpages of information about that resort.
Design by Michelangelo
In this project we were required to create a site that advertised an interior design company. Using HTML and CSS the tasks were to use columns to display "before", "during", and "after" photos of the company's work.

Cupcakes
In this project we were required to create a site that advertised a cupcake shop. Using HTML the tasks were to add photos and links to the website.
